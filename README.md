
# 🎭 Система актеров v1.0 — Custom Menu

Управление актерами в сцене: создание, переключение, перемещение, вращение, взаимодействие и заморозка персонажа.

---

## 🎮 Основное управление

| Клавиша           | Действие                                          |
|-------------------|---------------------------------------------------|
| `M`               | Открыть главное меню управления актерами          |
| `1` / `2`         | Переключение между актерами (предыдущий / следующий) |
| `3` / `4`         | Листание страниц скинов при создании актера       |
| `L`               | Удалить выбранного актера                         |
| `Z`               | Взаимодействие с актером (актер говорит своё сообщение) |
| `H`               | Включить / выключить режим перемещения актера     |
| `5`               | Заморозить себя (отключить управление персонажем) |
| `6`               | Разморозить себя (включить управление персонажем) |

---

## 🚶 Режим перемещения актера

> Включается клавишей `H` или через меню.

| Клавиша           | Действие                          |
|-------------------|-----------------------------------|
| `W`               | Двигать актера вперёд             |
| `S`               | Двигать актера назад              |
| `A`               | Двигать актера влево              |
| `D`               | Двигать актера вправо             |
| `Пробел`          | Поднять актера вверх              |
| `Ctrl`            | Опустить актера вниз              |
| `Q`               | Повернуть актера налево           |
| `E`               | Повернуть актера направо          |

---

## 📝 Примечания

- Режим перемещения актера должен быть активен (`H`), чтобы двигать или вращать актера.
- При создании актера можно листать скины клавишами `3` / `4`.
- Заморозка себя (`5`) полностью отключает управление персонажем — полезно для тестирования актеров.
- Взаимодействие (`Z`) заставляет актера произнести своё уникальное сообщение.
